TO RUN the program :

1. First you need to run 'UDPServer.py' in a terminal : $ python UDPServer.py
2. Then you need to run 'UDPClient.py' into a different terminal window : $ python UDPClient.py
3. Normally you should have :

In UDPServer terminal you should see :  ` b'Hello UDP server ! I am the Client !' `
In UDPClient terminal you should see :  ` Server says b'Hello I am UDP server' `
